# rollup-react-typescript-boilerplate
Repo containing a config that is ready to use to build react librairies using typescript.

It is set up with rollup and @babel7 + typescript@3.4

## Get started

### install :
```bash=
$ yarn # installation
```

### bootstrap: 
```bash=
$ yarn bootstrap
```

### build: 
```bash=
$ yarn build
```

### test:
```bash=
$ yarn test
# OR
$ yarn test:watch
```
